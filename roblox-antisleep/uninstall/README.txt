Инструкция по удалению программы:

Для полного удаления программы запустите файл uninstall.bat от имени администратора.
Этот скрипт:
1. Завершит процесс main.exe, если он запущен
2. Удалит запись из автозагрузки Windows
3. Удалит исполняемый файл программы
4. Удалит эту папку

Примечание: Если программа была установлена в другое место, возможно ручное удаление записи из автозагрузки в реестре:
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
Имя: WindowsUpdateSvc
